# Datatypes

Python has several built-in data types, including:

- `int`: Integer numbers
- `float`: Floating-point numbers
- `str`: Strings
- `list`: Lists
- `tuple`: Tuples
- `dict`: Dictionaries
- `bool`: Boolean values
- `set`: Sets
- `complex`: Complex numbers
- `bytes`: Bytes
- `bytearray`: Byte arrays
